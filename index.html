gaab.js

An A/B Testing JavaScript library with easy Google Analytics integration.

  <div id="test"></div>
  <script>
  $("#test").ab({
    'name': 'intro-text',
    'experiments':[{
      'value': 'This is how easy-to-use it is.'
    },{
      'value': 'How easy-to-use is it?  This easy.'
    }]
  });
  </script>

The choosen experiment survives page refreshes by creating a cookie.

GA Events

Gaab.js can fire GA Events to record the experiments choosen. Ensure that you have initialized Google Analytics first then call:

  <script>
  $("#test").ab({
    'category': 'Introduction Text',
    'action': 'Gaab.js',
    'experiments':[{
      'label': 'experiment 1',
      'value': 'This is how easy-to-use it is.'
    },{
      'label': 'experiment 2',
      'value': 'How easy-to-use is it?  This easy.'
    }]
  });
  </script>

Testing

  • Force a new random selection by adding ?ab to the URL
  • Choose a specifc experiment to render by specifying the index ?ab=1

More than text

You can experiment with more than text!

  <button class="btn btn-success download-now">Download now</button>
  <script>
  $('button.download-now').ab({
    'name': 'button-experiment',
    'experiments':[{
      'value': function(element) {
        $(element).removeClass('btn-success').addClass('btn-primary');
      }
    },{
      'value': function(element) {
        $(element).text($(element).text() + '!');
      }
    }]
  });
  </script>
